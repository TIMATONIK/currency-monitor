# 💱 Currency Monitor

Веб-утилита мониторинга курсов валют на основе данных Центрального банка РФ.

---

## Описание

Минималистичный однофайловый инструмент для отслеживания курсов USD, EUR и CNY к рублю. Позволяет задать пороговый курс и мгновенно получить визуальный алерт — выше или ниже порога находится текущее значение. Данные берутся напрямую с API ЦБ РФ в реальном времени.

---

## Функционал

- Получение актуального курса с [cbr-xml-daily.ru](https://www.cbr-xml-daily.ru/daily_json.js)
- Выбор валюты: **USD**, **EUR**, **CNY**
- Ввод порогового курса (необязательно)
- Отображение текущего курса и отклонения от порога
- Цветовые алерты:
  - 🔴 Курс **выше** порога
  - 🟢 Курс **ниже** порога
  - 🟡 Курс **равен** порогу
- История последних **5 запросов** с индикаторами статуса
- Пульсирующий индикатор актуальности данных

## Как запустить

### Вариант 1 — открыть напрямую

Скачайте `index.html` и откройте его в любом современном браузере:

```
Файл → Открыть → index.html
```

> Браузер должен разрешать CORS-запросы к внешним API. В большинстве браузеров при открытии через `file://` это работает корректно.

### Вариант 2 — локальный сервер

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем откройте [http://localhost:8080](http://localhost:8080).

### Вариант 3 — GitHub Pages

Репозиторий готов к деплою через GitHub Pages:

1. Settings → Pages
2. Source: `main` / `/(root)`
3. Сохранить — страница будет доступна по адресу `https://timatonik.github.io/currency-monitor`

---

## API

Используется публичный API ЦБ РФ без ключа:

```
GET https://www.cbr-xml-daily.ru/daily_json.js
```

Данные обновляются ежедневно в рабочие дни.
