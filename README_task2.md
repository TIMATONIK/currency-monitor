# Currency Parser — ЦБ РФ

Консольный Python-скрипт для получения и сохранения курсов топ-10 валют с API Центрального банка РФ.

---

## Описание

Скрипт запрашивает актуальные курсы, вычисляет стоимость за единицу каждой валюты, выводит отсортированную таблицу в консоль и сохраняет результат в CSV-файл с датой в названии (`rates_ДД-ММ-ГГГГ.csv`).

**Валюты:** USD, EUR, CNY, GBP, JPY, CHF, HKD, SGD, TRY, KZT

---

## Запуск локально

```bash
pip install -r requirements.txt
python currency_parser.py
```

---

## Запуск в Docker

### Сборка образа

```bash
docker build -t currency-parser .
```

### Запуск (таблица в консоли)

```bash
docker run --rm currency-parser
```

### Запуск с сохранением CSV на хост

```bash
docker run --rm -v "$(pwd)/output:/app" currency-parser
```

CSV-файл появится в папке `output/` рядом с проектом.

## Обработка ошибок

| Ситуация | Поведение |
|---|---|
| API недоступен | Сообщение в stderr, exit code 1 |
| Таймаут (>10 сек) | Сообщение в stderr, exit code 1 |
| HTTP-ошибка | Код ответа в stderr, exit code 1 |
| Неверный JSON | Сообщение в stderr, exit code 1 |
| Валюта отсутствует | Предупреждение, остальные обрабатываются |
