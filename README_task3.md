# Currency Rates — Google Apps Script

Скрипт для Google Таблиц: автоматически загружает курсы 10 валют с API ЦБ РФ и записывает их в лист с форматированием.

---

## Как подключить

1. Откройте Google Таблицу.
2. **Расширения → Apps Script**.
3. Удалите содержимое файла `Код.gs` и вставьте содержимое `currency_rates.gs`.
4. Сохраните (`Ctrl+S`).
5. Закройте и снова откройте таблицу — в меню появится пункт **«Валюты ЦБ РФ»**.
6. Нажмите **«Обновить курсы»** — при первом запуске браузер попросит разрешить доступ к внешним URL.

---

## Триггер

Скрипт использует **простой триггер `onOpen()`** — он запускается автоматически каждый раз, когда пользователь открывает таблицу, и добавляет пункт меню. Установка вручную не требуется.

Сам запрос к API происходит только по явному действию пользователя (клик «Обновить курсы»).

### Опционально: автоматическое обновление по расписанию

Чтобы курсы обновлялись без участия пользователя (например, ежедневно в 9:00):

1. В редакторе Apps Script → **Триггеры** (значок часов).
2. **+ Добавить триггер**.
3. Функция: `updateRates`, событие: **По времени → День → 9:00–10:00**.

## API

| Параметр | Значение |
|---|---|
| URL | `https://www.cbr-xml-daily.ru/daily_json.js` |
| Метод | GET |
| Формат ответа | JSON |
| Обновление данных | Ежедневно в рабочие дни |
| Авторизация | Не требуется |

Пример структуры ответа:
```json
{
  "Date": "2026-05-16T11:30:00+03:00",
  "Valute": {
    "USD": { "Name": "Доллар США", "Nominal": 1, "Value": 73.1275, ... },
    "JPY": { "Name": "Иен",       "Nominal": 100, "Value": 46.14, ... }
  }
}
```

> `Value` — курс за `Nominal` единиц. Курс за единицу: `Value / Nominal`.

## Логика обработки

```
onOpen()
  └─ Добавляет меню «Валюты ЦБ РФ → Обновить курсы»

updateRates()
  ├─ UrlFetchApp.fetch(API_URL, { muteHttpExceptions: true })
  │     muteHttpExceptions — ошибки HTTP не бросают исключение,
  │     позволяет проверить getResponseCode() вручную
  │
  ├─ Проверка HTTP-кода (ожидается 200)
  ├─ JSON.parse() в try/catch — ловим невалидный ответ
  ├─ Проверка наличия поля Valute в структуре
  │
  ├─ Цикл по CODES → для каждой валюты:
  │     rate = Value / Nominal
  │     пропуск с Logger.log() если код отсутствует в ответе
  │
  ├─ Сортировка строк по rate DESC
  │
  ├─ Запись в лист:
  │     A1        — статус (успех/ошибка) + цветовой фон
  │     A2:E2     — заголовки (жирные, тёмный фон)
  │     A3:E12    — данные, колонка E — формат #,##0.0000
  │     чередующийся фон строк, autoResizeColumn
  │
  └─ catch(err) → A1 красный статус, Logger.log()
```

---

## Обработка ошибок

| Ситуация | Поведение |
|---|---|
| API недоступен / таймаут | `UrlFetchApp` бросает исключение → `catch` → красный статус в A1 |
| HTTP-код ≠ 200 | Ручная проверка → `throw new Error('HTTP XXX')` → красный статус |
| Ответ не является JSON | `JSON.parse()` в `try/catch` → понятное сообщение |
| Нет поля `Valute` | Проверка структуры → `throw` с описанием |
| Отдельная валюта отсутствует | Предупреждение в `Logger`, остальные записываются |
